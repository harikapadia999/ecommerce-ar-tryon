# 🛍️ E-commerce AR Virtual Try-On

> AR product visualization and virtual fitting room for online shopping

[![Live Demo](https://img.shields.io/badge/demo-live-success)](https://harikapadia999.github.io/ecommerce-ar-tryon/)

## ✨ Features

### 👗 Virtual Try-On
- **Clothing AR** - See how clothes fit in real-time
- **Accessories** - Try watches, jewelry, glasses
- **Footwear** - Visualize shoes on your feet
- **Makeup** - Virtual makeup application

### 🏠 Furniture Placement
- **AR Room Preview** - Place furniture in your space
- **Size Visualization** - See actual dimensions
- **Color Variations** - Try different colors/materials
- **360° Product View** - Rotate and inspect items

### 🎨 Customization
- **Color Picker** - Change product colors
- **Material Selection** - Wood, metal, fabric options
- **Size Comparison** - Compare different sizes
- **Style Matching** - AI-powered style suggestions

## 🛠️ Tech Stack

- React 18.3 + TypeScript
- Three.js + React Three Fiber
- WebXR for AR support
- AR.js for marker-based AR
- Tailwind CSS

## 🚀 Quick Start

```bash
npm install
npm run dev
```
